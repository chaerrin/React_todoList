 ## 음료수 자판기
  #### 기능 설명
   1. 동전을 투입한 후 원하는 음료수를 선택합니다. <br />
   2. 음료수 선택을 완료하면 거스름돈 버튼을 눌러 거스름돈을 반환합니다. <br />
   3. 투입한 금액보다 선택한 음료수의 가격이 크면 잔액이 부족하다는 경고창이 뜹니다. <br />
   4. 거스름돈을 확인하고 버튼을 누르면 구매한 음료수가 구매 품목으로 넘어갑니다.
   
  #### 프로젝트 정보
   제작 기간 : 3주 <br />
   참여 인원 : 1명 <br />
   사용 기술 : html, css, javascript
